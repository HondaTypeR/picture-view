## 简介
全屏查看组件

## 如何运行

```
安装依赖：
npm install

运行：
npm start
```

## 配置项

请注意，需要在组件外部传入控制组件显示的变量与改变变量的方法。

+ picView: bool, 是否显示的变量
+ picIndex: number, 显示的首张索引
+ toggleView: func, 控制picView的方法
+ pictures: array, 图片数组
+ items: array, 图片地址数组
+ isSwipe: bool, 是否可拖动, 默认false
+ hasArrow: bool, 是否显示箭头,默认true
+ hasDot: bool, 是否显示点，默认false
